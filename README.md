# Requirement
- PHP (8.1 or higher)
- Mysql
- Composer

# Installation Guide
- Clone Project
    ``` bash
    https://github.com/basiooo/waizly-assessment-backend.git
    ```
- Navigate to project dir
    ``` bash
    cd waizly-assessment-backend
    ```
3. Install project dependencies:

    ```bash
    composer install
    ```
